omega-s.az

Omega Service

Веб-сайт омега состоит из двух частей: пользовательская часть и административная. В пользовательской(это то что видит пользователь посещая данный веб-сайт) пользователь получает доступ к основной информации компании, поиску в каталоге и непосредственно доступ к каталогу с полным описанием каждого товара. В админ части сотрудники компании имеют функционал по управлению всем товаром, с возможностью указанием цен(фабричная и продажная), наличия на складе элементов. При появлении уже существующего товара на сайте его не нужно заносить в каталог опять, достаточно просто выставить число которое уже есть на складе. Весь функционал веб-сайта завязан на базу данных и ее манипуляцию.

isticker.az

Продажа стикеров, обоев, наклеек

Проект iSticker.az создавался с целью продвижения продаж данной компании. Данный веб-сайт является сайтом-визиткой. Данный веб-сайт выполнен в виде блоков, каждый из которых выполняет функцию описание того или иного типа товаров. Кроме блоков сайт обладает небольшим каталогом товаров без завязки на базу данных, которая для данного проекта не нужна в связи с небольшим количеством различных товаров.
